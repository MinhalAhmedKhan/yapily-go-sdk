# Backup

Secondary URL where events will be sent whenever the primary URL is not responding. It must not contain any query parameters.


## Fields

| Field                                 | Type                                  | Required                              | Description                           |
| ------------------------------------- | ------------------------------------- | ------------------------------------- | ------------------------------------- |
| `URL`                                 | `*string`                             | :heavy_minus_sign:                    | A valid URL without query parameters. |
